# Project Gutenbergが紡ぐ知の回廊：デジタル時代の古典アーカイブはなぜ今も重要なのか

Project Gutenbergは、50年を超える歴史の中で、人類の共有知をデジタル空間に刻み続けてきた。その活動が今なお新鮮な関心を集めるのはなぜか。単なる過去の遺産の保存以上の、見えない価値の回路がそこにはあるのかもしれない。私たちは、その淵源と持続のメカニズムを静かに観察している。
